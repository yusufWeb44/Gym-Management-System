Gym Management System is a full-stack web application designed to streamline gym operations and provide intelligent insights for administrators.

The system allows managing members, trainers, subscription plans, and fitness types through a clean and modern interface. It includes a smart dashboard that goes beyond basic CRUD operations by offering real-time analytics such as subscription alerts, daily gym activity, and trainer workload distribution.

Key highlights:
- Member and subscription management
- Trainer assignment and workload tracking
- Smart dashboard with analytics and insights
- Subscription status monitoring (Active, Expiring, Expired)
- Monthly registration tracking
- Secure authentication system (admin-only access)
- Clean UI with a modern gym-themed design

Built using ASP.NET Core MVC, Entity Framework Core, SQL Server, Bootstrap, and Chart.js.
